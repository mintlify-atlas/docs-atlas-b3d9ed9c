Vaishali Visions is a personal portfolio and creative showcase built with React 18, TypeScript, Vite, and Tailwind CSS. It features a soft lavender-peach aesthetic, glassmorphism cards, scroll-reveal animations, and seven fully designed sections — from a hero banner to a contact form. These docs cover every section, how to customize the design, and how to run and deploy the project.

## Quick start

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/gladis-git-bit/vaishali-visions.git
    cd vaishali-visions
    ```
  </Step>
  <Step title="Install dependencies">
    <CodeGroup>
    ```bash npm
    npm install
    ```
    ```bash bun
    bun install
    ```
    </CodeGroup>
  </Step>
  <Step title="Start the dev server">
    <CodeGroup>
    ```bash npm
    npm run dev
    ```
    ```bash bun
    bun run dev
    ```
    </CodeGroup>
    Open [http://localhost:5173](http://localhost:5173) in your browser.
  </Step>
  <Step title="Personalize your content">
    Edit the component files in `src/components/` to update your name, bio, projects, and more. See the [Customization guides](/customization/updating-content) for details.
  </Step>
</Steps>
