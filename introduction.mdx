---

Vaishali Visions is a single-page portfolio application built with React 18, TypeScript, and Vite. It ships a cohesive dreamy aesthetic — glassmorphism cards with frosted-glass backdrops, animated floating blobs, and a soft lavender-peach-butter color system anchored by a lavender primary (`hsl(268 60% 70%)`, approximately `#c084e8`), a peach accent, and a hot-pink pop color for high-contrast moments. The app is organized into seven distinct sections — Hero, About, Skills, Experience, Projects, Blog, and Contact — that scroll into view with CSS-driven reveal animations. The component library is built on Radix UI primitives via shadcn/ui, with TanStack Query managing client state, React Router DOM handling in-app routing, and Recharts for any data visualizations. Styles are written in Tailwind CSS, and the three typefaces — DM Serif Display for headings, Inter for body text, and Caveat for handwritten script accents — are loaded from Google Fonts.

## Key technologies

  <Card title="Vite" icon="zap" href="https://vitejs.dev">
    Build tool and dev server using the SWC-based React plugin for fast HMR and optimized production bundles.
  </Card>

    Accessible component primitives built on Radix UI, including dialogs, tooltips, toasts, and more.

</Note>
